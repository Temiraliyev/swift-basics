# Swift чист?

**Swift** — забони муосири барномасозӣ мебошад, ки аз ҷониби Apple соли 2014 сохта шудааст.

Ин забон барои сохтани:

* iOS applications
* macOS applications
* watchOS applications
* tvOS applications

истифода мешавад.

Swift забони:

* fast
* safe
* modern
* powerful

ҳисобида мешавад.

Swift бар асоси LLVM compiler сохта шудааст ва performance-и хеле баланд дорад. Инчунин бо Objective-C пурра interoperable мебошад ва метавонад бо UIKit ва дигар Apple frameworks кор кунад.

Яке аз бартариҳои муҳимтарини Swift:

* type safety
* optional system
* memory safety
* compile-time error checking

мебошад, ки барои навиштани коди устувор ва бехатар кӯмак мекунад.
